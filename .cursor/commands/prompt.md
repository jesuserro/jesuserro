# Crear Prompt de Ayuda Completo

Genera un prompt completo para solicitar ayuda a GPT, documentando todo el problema, contexto, soluciones posibles, código relevante, estructura de archivos, intentos realizados y todas tus dudas.

## Descripción

Este comando crea un prompt estructurado y completo que incluye:
- Documentación detallada del problema
- Contexto completo del proyecto
- Posibles soluciones consideradas
- Código real relevante
- Estructura de archivos y carpetas (tree)
- Intentos realizados previamente
- Todas las dudas y preguntas específicas

## Uso

El comando generará un archivo `.md` con un prompt completo que puedes usar directamente con GPT para obtener ayuda técnica detallada y contextualizada.

## Características

- **Documentación completa**: Incluye todo el contexto necesario
- **Código real**: Incorpora código relevante del proyecto
- **Estructura clara**: Organiza la información de manera lógica
- **Preguntas específicas**: Incluye todas las dudas técnicas
- **Formato markdown**: Fácil de leer y usar
